# ol元素

## 有序列表

ol：ordered list
li：list item

```html
把大象装冰箱，总共分几步？
<ol>
    <li>打开冰箱</li>
    <li>把大象放进去</li>
    <li>关上冰箱</li>
</ol>
```
ol元素有个reversed（boolean）属性，设置后表示列表倒置显示
页面效果:
```html
把大象装冰箱，总共分几步？

关上冰箱
把大象放进去
打开冰箱
```

## 无序列表
ul：unordered list
无序列表常用于制作菜单、新闻列表等
```html
<ul>
    <li>运动类</li>
    <li>美妆类</li>
    <li>电子类</li>
</ul>
```
